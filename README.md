# Resolvendo Codigos py com-Auxilio da IA - DIO
No curso de GitHub da DIO, é utilizado o GitHub Copilot, mas como ainda não tenho acesso, utilizarei o chat GPT como auxiliar nessa aula



1 - Concatenando Dados

Descrição: Vamos receber dois dados diferentes do usuário e concatena-los em uma única string ?!

O que aprenderemos?

· Manipulação de Strings (string)
· Concatenação
· Entrada de dados
· Utilização eficiente do Github Copilot

2 - Repetindo Textos

Descrição: Agora vamos solicitar uma string e um número inteiro como entrada. Depois teremos que retornar a string
repetida o número de vezes informado.

O que aprenderemos?

· Manipulação de Strings (string)
· Números Inteiros (int)
· Múltiplas repetições
· Entrada de dados
· Aproveitar as sugestões do Github Copilot

3 - Operações Matemáticas Simples

Descrição: Vamos solicitar como entrada dois números e depois vamos realizar uma operação simples entre eles.

O que aprenderemos?

· Operações Matemáticas Básicas
· Entrada de dados
· Utilização eficiente do Github Copilot
